# Week 1 — HTML Fundamentals

**Theme:** Foundations (HTML & CSS)  
**Suggested time:** 12–16 hours total

---

## Learning goals

- Semantic HTML structure
- Forms and input validation
- Accessibility basics (alt text, ARIA)
- SEO-friendly structure
- Code organization and best practices

## Practice projects

- [ ] Personal “About Me” page
- [ ] Recipe page with semantic structure (header, nav, article, section, footer)
- [ ] Multi-page site: navbar, footer, forms

---

## What to focus on

- **Semantic tags:** `header`, `nav`, `article`, `section`, `aside`, `footer`
- **Forms:** inputs, labels, validation attributes, `required`, `pattern`
- **Accessibility:** `alt` on images, ARIA labels where needed
- **SEO:** logical headings (h1 → h2 → h3), meta description
